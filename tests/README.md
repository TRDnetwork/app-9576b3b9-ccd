# Portfolio Pro - Test Suite

## How to Run
1. Install dependencies: `npm install vitest jsdom`
2. Run tests: `npm test`

## Test Coverage
- `app.test.js`: Unit tests for theme toggle logic, smooth scrolling behavior, and DOM initialization sequence.
- `api.test.js`: Confirms no external API usage and validates reliance on localStorage for theme persistence.

All tests verify core functionality of the static portfolio site as specified.
